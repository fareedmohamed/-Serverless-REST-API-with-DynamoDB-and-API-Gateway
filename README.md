# -Serverless-REST-API-with-DynamoDB-and-API-Gateway

**Architecture:** Serverless

## Description
Develop a serverless REST API using Amazon API Gateway, AWS Lambda, and DynamoDB to manage a simple to-do list or customer records. The API allows users to create, read, update, and delete (CRUD) data without managing servers.

## Key AWS Services Used
* **Amazon API Gateway:** Exposes REST endpoints.
* **AWS Lambda:** Handles API requests (CRUD operations).
* **Amazon DynamoDB:** NoSQL database for storing records.
* **AWS IAM:** Controls access via roles and permissions.
* **Amazon CloudWatch:** Logs and monitors API activity.
* **S3:** to host the front-end of your applicatio
